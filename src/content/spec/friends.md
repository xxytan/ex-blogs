## 🤔How to?
贵站如需交换友链，请以以下格式
- 在下方评论区留言🔗
- 或[给我邮件](mailto:msbyshens@outlook.com)📧，邮件主题就写`友链申请`
:::warning[若通过邮件投递申请]
请不要使用您的自定义域邮箱投递，<strong>100%</strong>会被自动归为垃圾邮件！
:::

```yaml
title: "站点标题",
imgurl: "站点图标",
desc: "站点描述",
siteurl: "站点链接"
```

*例如我的*
```yaml
title: "傲雪の",
imgurl: "https://avatars.githubusercontent.com/u/155360005",
desc: "无聊的一天搭配神经的我～",
siteurl: "https://b.oxue.de"
```

## 👀提醒
- 站长可能因为某些原因（上学、睡觉、吃席…）未能及时添加友链，但不会装作看不见🤫
- 请确保你的站点处于**可用**状态，站长会偶尔查看，若发现您的网站无法访问，您的友链会被打上`失联`标签，直至您的站点恢复正常🥰
- 请不要单方面友链，站长发现将会**删除您的友链**！😠
- 若站长已答应添加友链，但仍找不到自己的网站❓
  1. 网站正在构建中…<p align="right"><strong><i>请耐心等待…</i></strong></p>
  2. 网站因错误配置导致构建失败<p align="right"><strong><i>耐心等待站长修复错误并重新部署站点…</i></strong></p>
  3. 网站因不可抗力因素构建失败<p align="right"><strong><i>没招了💀</i></strong></p>
